# Curiosidades Modernas 🌟

Um site interativo de curiosidades do mundo!

## 🚀 Tecnologias

- HTML5 semântico
- SCSS (Sass)
- JavaScript (ES6+)
- Vite
- GitHub Pages

## 👨‍💻 Como rodar localmente

```bash
npm install
npm run dev
```

## 🔨 Build para produção

```bash
npm run build
```

## 🌐 Publicação

Use o GitHub Pages para publicar a pasta `dist/`.
